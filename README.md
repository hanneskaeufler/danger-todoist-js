> A danger plugin for spotting introduced todos.

## Usage

Install:

```sh
yarn add danger-plugin-danger-todoist-js --dev
```

At a glance:

```js
// dangerfile.js
import dangerTodoistJs from 'danger-plugin-danger-todoist-js'

dangerTodoistJs()
```
